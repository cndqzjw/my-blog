## SEO 结构要求

### 标题层级
- H1：含主关键词，格式：「Best [产品类型] in [年份]: [差异化角度] (Tested & Reviewed)」
- H2（产品）：每款产品一个，格式：「[序号]. [品牌型号] — [核心卖点短语]」
- H2：「How We Selected These [产品类型]」（E-E-A-T 信号，说明选品标准）
- H2：「[产品类型] Buying Guide」（含长尾关键词1，至少3个选购维度）
- H2：「Frequently Asked Questions」（含长尾关键词2，3-5个真实用户问题）

### 关键词分布规则
- 主关键词：H1、首段100字内、至少2个H2标题、meta description 中出现
- 长尾关键词1：自然融入 Buying Guide 段落
- 长尾关键词2：用作 FAQ 中某个问题的标题
- 关键词密度控制在 1-2%，禁止堆砌

### Front Matter SEO 字段
- title：≤60字符，必须含主关键词，以主关键词开头
- description：120-155字符，含主关键词，以行动动词开头（如 Discover / Find / Compare）
- slug：全小写英文，连字符分隔，含主关键词（如 best-drip-coffee-maker-2025）
- keywords：主关键词 + 长尾词1 + 长尾词2，最多5个

### 图片 SEO
- 每款产品配1张图片占位
- alt 文字格式：「[品牌型号] [产品类型] - [主要用途或特点]」
- 图片 URL 暂用占位：https://images-na.ssl-images-amazon.com/images/I/[占位].jpg

### 内容质量信号（E-E-A-T）
- How We Selected 段落：说明至少3个筛选标准（价格、用户评分、核心功能）
- 每款产品必须包含：核心参数 + 真实使用场景 + 一个明确缺点（增加可信度）
- Buying Guide 必须包含：选购维度标题（H3）+ 判断标准说明
- FAQ 问题来自该品类真实高搜索量疑问（如耐用性、清洁、兼容性）

### 联盟链接生成规则
为每款产品生成亚马逊搜索链接：
https://www.amazon.com/s?k=[关键词]&tag=cndqzjw20-20

关键词构造：
- 知道品牌型号 → k=品牌+型号（如 k=ninja+cfp307+coffee+maker）
- 只知道类型 → k=类型+2-3个核心特征（如 k=thermal+carafe+coffee+maker+programmable）
- 禁止生成任何 /dp/ASIN 格式直链
